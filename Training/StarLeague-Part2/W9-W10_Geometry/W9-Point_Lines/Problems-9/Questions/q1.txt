Sum of Manhattan Distances (Starleague Training Problem)

Given N (2 <= N <= 100,000) points with integer coordinates (-10,000 <= x, y <= 10,000). Calculate sum of 
Manhattan distances between all pairs of points.

INPUT FORMAT (file sumdist.in)
Line 1: N
Line 2 to N+1: Line i+1 contains xi and yi.

OUTPUT FORMAT (file sumdist.out)
An integer that is the sum of Manhattan distances. Output the answer modulo 1000000007.

SAMPLE INPUT
3
1 1
3 1
-1 -2

SAMPLE OUTPUT
14
